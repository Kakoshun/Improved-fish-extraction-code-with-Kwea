==> Using the Kwea tool for extracting fish from images 

Using the Kwea tool for extracting new images with fish from pictures. In the code I provide, the incoming image for processing is converted into a blue scale image, resulting in increased effectiveness of the Kwea tool. Compared to the original version of Kwea, converting the incoming image to a blue scale shows a dramatic increase in the production of images with fish.  
The code contains comments in both English and Greek.

==> Χρήση του εργαλείου Kwea για την εξαγωγή ψαριών από εικόνες

Χρήση του εργαλείου Kwea για την εξαγωγή νέων εικόνων με ψάρια από εικόνες. Στον κώδικα που παραθέτω, γίνεται μετατροπή της εισερχόμενης προς επεξεργασία εικόνας, σε εικόνα της κλίμακας του μπλε, με αποτέλεσμα να αυξάνεται η αποτελεσματικότητα του εργαλείου Kwea. Σε σύγκριση με την αρχική έκδοση του Kwea ή μετατροπή της εισερχόμενης εικόνας σε κλίμακα του μπλε, παρουσιάζει θεαματική αύξηση στην παραγωγή εικόνων με ψάρια. 
Ο κώδικας περιέχει σχόλια στα αγγλικά και στα ελληνικά.
